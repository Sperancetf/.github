## 📜 Community Guidelines / Reglamento

### [EN] Quick Rules
* **No Cheating:** Fair play is mandatory. Hardware bans apply for macros/cheats.
* **Privacy:** Zero tolerance for doxing or sharing personal data.
* **Integrity:** Report bugs; do not exploit them for personal gain.

### [ES] Reglas Rápidas
* **Sin Trampas:** Juego limpio obligatorio. Aplicamos hardware bans para macros o cheats.
* **Privacidad:** Tolerancia cero al doxing o difusión de datos personales.
* **Integridad:** Los errores se reportan, no se aprovechan.
